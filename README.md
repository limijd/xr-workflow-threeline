# xr-workflow-threeline

一个给 AI 辅助开发使用的“三线流” workflow skill：在做项目的同时，按需推进项目交付、领域学习和 AI 协作方法沉淀。

## What It Does

- 用 `THREELINE_PLAN.md` 统一驱动任务推进
- 把 `DL`（交付）、`DK`（领域知识）、`HE`（AI 驾驭）编织进同一条工作流
- 支持初始化、会话恢复、计划检查和中途切换模式

## When To Use

- 你要进入一个不熟悉的技术域或业务域
- 你想边做边学，而不是先学完再开工
- 你希望把 AI 协作经验持续沉淀到项目文档

## Trigger

- `/xr-workflow-threeline`
- `/三线战法`
- 或自然语言提到“三线流”“threeline”“边战边学”
